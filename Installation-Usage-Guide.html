DaVinci Resolve V2V 安装指南

步骤 1：检查 DaVinci Resolve 中是否已安装 Python

点击：工作区 ——> 控制台 ——> Py3。如果已安装，则不会弹出窗口（直接进入 步骤 2）。

若未安装，则会弹出 "Python was not found" 消息，请根据提示下载并安装，安装完成后进入步骤 2！

Windows 用户：安装 Python 时请务必勾选 "Add Python to PATH" 选项，否则可能需要重新安装或手动设置环境变量。

步骤 2：移动 DaVinci V2V 插件

将 DaVinci V2V 文件夹移动到以下位置：

Mac: 
/Library/Application Support/Blackmagic Design/DaVinci Resolve/Fusion/Scripts/Utility 

Win: 
C:\ProgramData\Blackmagic Design\DaVinci Resolve\Fusion\Scripts\Utility

步骤 3：安装依赖

Mac 用户：

双击 Mac_Install.command进行安装，等待安装完成。

Win 用户：

双击 Windows_Install.bat进行安装，等待安装完成。

安装完成后，可通过 工作台 ——> 脚本 菜单打开插件进行使用。

使用指南

Runway 配置

  • API Key： 必填，用于服务认证。
  • Base URL： 选填，用于第三方代理服务。不填则使用官方默认地址。

生成选项

  • Prompt： 视频生成的描述词。
  • 模型 (Model)： 选择生成视频所用的模型。
  • 比例 (Ratio)： 设置生成视频的宽高比。插件会根据所选比例自动创建渲染预设。
  • 随机种子 (Seed)： 用于控制生成结果的一致性。可手动输入，或勾选“随机”自动生成。
  • 参考图 (Reference)： 选择一张图片作为生成视频的风格或内容参考。

任务处理

  • 生成： 提交一个新的视频生成任务。插件会自动渲染时间线上的标记区域作为输入视频。
  • 下载视频： 如果已有任务ID，可在此输入并点击下载，无需重新提交。

其他设置

  • 保存路径： 设置一个目录，用于保存生成的视频文件。
  • 语言： 切换插件界面的显示语言（中文/英文）。

DaVinci Resolve V2V Installation Guide

Step 1: Check if Python is Installed in DaVinci Resolve

Click: Workspace ——> Console ——> Py3. If it is already installed, no window will pop up (proceed directly to Step 2).

If it is not installed, a "Python was not found" message will pop up. Follow the pop-up instructions to download and install it. After installation, proceed to Step 2!

Important for Windows users: During Python installation, make sure to check the "Add Python to PATH" option. If you forget this step, you may need to reinstall Python or manually add it to the system's environment variables.

Step 2: Move the DaVinci V2V Plugin

Move the DaVinci V2V folder to the following location:

Mac: 
/Library/Application Support/Blackmagic Design/DaVinci Resolve/Fusion/Scripts/Utility 

Win: 
C:\ProgramData\Blackmagic Design\DaVinci Resolve\Fusion\Scripts\Utility

Step 3: Install Requirement

For Mac Users:

Double-click the Mac_Install.command ,Install it and wait for it to complete..

For Windows Users:

Double-click the Windows_Install.bat ,Install it and wait for it to complete..

After installation, open the plugin in DaVinci Resolve by navigating to Workspace ——> Scripts.

Usage Guide

Runway Configuration

  • API Key: Required for service authentication.
  • Base URL: Optional, for third-party proxy services. If left blank, the official default address will be used.

Generation Options

  • Prompt: The descriptive text for video generation.
  • Model: Select the model to be used for video generation.
  • Ratio: Set the aspect ratio of the generated video. The plugin will automatically create a render preset based on the selected ratio.
  • Seed: Used to control the consistency of the generation results. Can be entered manually or generated randomly by checking "Random".
  • Reference: Select an image to be used as a style or content reference for the video generation.

Task Handling

  • Generate: Submit a new video generation task. The plugin will automatically render the marked region on the timeline as the input video.
  • Download Video: If you already have a Task ID, you can enter it here and click to download, without resubmitting.

Other Settings

  • Save Path: Set the directory where the generated videos will be saved.
  • Language: Switch the display language of the plugin interface (Chinese/English).
